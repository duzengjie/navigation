# 何如使用
更改每个Tab页面的数据 位于public里面的data.json文件

# 本地运行
```shell
cd ./navigation
```

```shell
pnpm install
```

```shell
pnpm run dev
```

# 部署
1. 修改.env.production
2. 打包
```shell
pnpm run build
```
